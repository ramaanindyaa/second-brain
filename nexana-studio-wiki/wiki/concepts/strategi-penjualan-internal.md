# Strategi Penjualan (Internal) — Nexana Studio

> **INTERNAL ONLY.** Sumber menandai eksplisit: "Jangan Tampilkan ke Klien."
> Halaman ini untuk referensi diri sendiri saat presentasi/closing, bukan
> materi proposal atau website.

Status: **decided** sebagai pedoman kerja internal (Nexana_Studio_Pricelist.md).

## 4 Taktik

1. **Anchoring.** Saat presentasi ke klien, tunjukkan paket PRO dulu (meski
   klien jelas tidak butuh), baru turun ke Growth/Starter. Paket termahal
   bikin harga di bawahnya kelihatan masuk akal.
2. **Jangan tulis "Mulai dari Rp..." tanpa angka atas.** Range harga (bukan
   "mulai dari") bikin klien tidak parno dipermainkan saat nego harga final.
3. **Tawarkan retainer di momen closing paket Starter/Growth**, bukan
   setelah proyek selesai — saat itu klien paling antusias dan gampang bilang
   "iya" untuk hal tambahan.
4. **Breakdown target Rp5jt/bulan:** 2 project Starter, ATAU 1 Growth +
   beberapa retainer, ATAU 1 Custom kecil. Begitu ada 5-8 klien retainer
   aktif, sebagian target bulanan sudah "aman" bahkan sebelum ada project
   baru.
